# 📊 Madhav E-Commerce Sales Dashboard (Power BI)

## 📌 Project Overview

This project is an interactive **Power BI Dashboard** developed to analyze the sales performance of an e-commerce business. The dashboard provides insights into sales, profit, customer behavior, product categories, payment methods, and regional performance through dynamic visualizations and interactive filters.

The goal of this project is to transform raw sales data into meaningful business insights that help stakeholders make data-driven decisions.

---

## 🎯 Objectives

- Analyze overall sales performance.
- Monitor monthly profit trends.
- Identify top-performing states and customers.
- Compare product category and sub-category performance.
- Analyze payment mode preferences.
- Create an interactive dashboard for business reporting.

---

## 🛠️ Tools & Technologies

- Microsoft Power BI
- Power Query
- DAX (Data Analysis Expressions)
- Data Modeling
- CSV Dataset

---

## 📊 Dashboard Features

- 📈 KPI Cards
  - Total Sales Amount
  - Total Profit
  - Total Quantity Sold
  - Average Order Value (AOV)

- 📅 Quarter-wise filtering using slicers

- 📊 Monthly Profit Analysis

- 🌍 State-wise Sales Analysis

- 👤 Customer-wise Sales Performance

- 🛍️ Category-wise Quantity Distribution

- 💳 Payment Mode Distribution

- 📦 Profit Analysis by Product Sub-Category

---

## 📈 Key Business Insights

- Generated **₹438K** in total sales.
- Earned an overall **₹37K** profit.
- Sold over **6K** products.
- Average Order Value (AOV) of **₹121**.
- Maharashtra contributed the highest sales.
- Clothing category accounted for the largest quantity sold.
- Cash on Delivery (COD) was the most preferred payment method.
- Printers generated the highest profit among all product sub-categories.
- Profit showed seasonal fluctuations, with strong performance in January and December.
